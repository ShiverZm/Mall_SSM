# 1 项目主题

# onlineShop
2018 基于Spring,SpringMvc,Mybatis,Jsp开发的购物商城，带后台及Security权限管理

## 1.1主题

基于SSM的购物网站

## 2 功能说明
### 主要具备以下功能：

#### 卖家：
##### （1）登陆
##### （2）发布商品
##### （3）管理商品
##### （4）查看订单
##### （5）查看销售信息
#### 会员：
##### （1）登陆注册
##### （2）浏览商品
##### （3）购买结算
##### （4）生成订单
